Hablei Assistant is an end-to-end local AI voice assistant built with Faster-Whisper, Ollama, and Coqui XTTS. The pipeline performs speech-to-text (ASR), large language model inference, and text-to-speech (TTS) with voice cloning, all running locally.

This project was developed to deepen my understanding of the theoretical concepts behind Audio Processing with Deep Learning, including Automatic Speech Recognition (ASR), Large Language Models (LLMs) and Text-to-Speech (TTS). It explores how these technologies can be integrated into a complete local speech processing pipeline.
